# Number Guess Game (C++)

This is a simple number guessing game written in C++.

The program generates a random number and the user tries to guess it.  
The program tells the user whether the guess is too high, too low, or correct.

## Features
- Random number generation
- User input
- Feedback for each guess
- Simple console interface

## Technologies
- C++
- Visual Studio
